# yarn

## 安装yarn

1. https://yarnpkg.com/zh-Hant/docs/install#windows-stable

2. npm i -g yarn 


如果使用yarn 就不能使用npm

## 安装所有依赖
```shell
yarn install
```

## 安装单个依赖
```shell
yarn add module_name
```

### 安装开发依赖
```shell
yarn add module_name -D
```


### 删除依赖
```shell
yarn remove module_name -D
```
